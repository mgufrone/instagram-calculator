## Instagram Engagement Calculator

### Requirements
- Golang
- NodeJS
- Yarn
- CMake

### Running the web app
Before anything else, set it up by running:
```shell
make setup
```
It will download dependencies for each app (server & ui)

Then, run this command in a separate terminal
For web server
```shell
make serve
```
For web UI
```shell
cd ui && yarn serve
```
